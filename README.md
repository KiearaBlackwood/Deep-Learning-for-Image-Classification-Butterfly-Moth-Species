# Deep-Learning-for-Image-Classification-Butterfly-Moth-Species

Python · TensorFlow · Keras · CNN · Matplotlib · Jupyter Notebook

- Designed and trained a Convolutional Neural Network (CNN) to classify images across 100 butterfly and moth species using a labeled image dataset.
- 
- Applied an 80/20 train-test split and tuned model architecture (layers, activation functions, dropout) to improve classification accuracy and reduce overfitting.
- 
- Evaluated model performance using accuracy and loss curves across training epochs, iterating on architecture to improve generalization on held-out test data.
